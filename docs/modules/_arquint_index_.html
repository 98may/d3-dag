Module "arquint/index"

This layout algorithm treats nodes not as points (i.e. producing x & y coordinates) but as rectangles. An accessor is supplied to extract a heightRatio from each node, specifying its height in comparison to other nodes. The implementation was contributed by the author L. Arquint and provides different algorithms to distribute the nodes along the x-axis.
